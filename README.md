# odin-etch-a-sketch
Repository for The Odin Project's Etch-a-Sketch project.

The project's goal is to build a browser version of an Etch-a-Sketch using DOM manipulation and requires the right JavaScript methods and CSS to complete. 

More information on the Etch-a-Sketch project can be found at The Odin Project website https://www.theodinproject.com/lessons/foundations-etch-a-sketch.
