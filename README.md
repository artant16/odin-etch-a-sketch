# odin-etch-a-sketch

Repository for The Odin Project's **Etch-a-Sketch** project.

### Overview
The project's goal is to build a browser version of an etch-a-sketch using DOM manipulation and requires using the right JavaScript methods and CSS to complete. 

More information can be found at [Project: Etch-a-Sketch](https://www.theodinproject.com/lessons/foundations-etch-a-sketch).
